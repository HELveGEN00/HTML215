Требования к экзаменационному сайту по курсу "XHTML, HTML, CSS"

По курсу "XHTML, HTML, CSS" необходимо сделать сайт, который должен быть доступен в интернете. В качестве результата студентом должна быть предоставлена ссылка на сайт.

Варианты: сайт-портфолио, фан-сайт (человека, группы, медиа-продукта, города, страны), новостной сайт, сайт-визитка, фотогалерея, сайт-каталог, блог, интернет-магазин

Сайт не принимается к рассмотрению, если не выполнены следующие обязательные требования:

  1. Шесть или больше связанных ссылками веб-страниц. Файл главной страницы называется index.html
  2. Использовать только блочную верстку (элементы <div>). Не использовать табличную верстку (элементы <table>, <tr>, <td>)
  3. Не использовать фреймы (кроме элемента iframe — его можно), физические элементы и атрибуты: тип всех XHTML документов должен быть XHTML 1.0 Strict или HTML5
  4. Использовать технологию стилей, все стили в отдельном CSS-файле, не использовать встроенные стили (атрибут style)
  5. Все документы XHTML и CSS должны быть валидными (валидатор XHTML, валидатор CSS)
  6. Не должно быть ссылок, ведущих на несуществующие страницы. Не должно быть недоступных изображений. Не должно быть проблем с кодировкой

Обязательные требования, за невыполнение которых оценка будет снижена:

  1. Кодировка XHTML документов должна быть явно задана с помощью элемента <meta>.
  2. Главное и дополнительные меню в виде маркированных списков (элемент <ul>), отображаются в одних и тех же местах на всех страницах (исключение может составлять только главная страница, на которой меню можно разместить в отличном от других страниц месте)
  3. Использовать графику (изображения в контенте и/или фоновые изображения)
  4. Сайт должен быть читаем и удобен в использовании при выключенных картинках. У всех элементов <img /> должны быть адекватные, понятные значения атрибутов alt, кратко описывающие изображение
  5. Сайт должен быть читаем и удобен в использовании при выключенных стилях
  6. Сайт должен быть читаем и удобен в использовании при ширине окна браузера 1024px и более. Недопустимо появление горизонтальной полосы прокрутки для вертикально-ориентированного сайта
  7. На главной странице внизу должна быть информация о вас: ФИО, номер группы, год создания сайта. Например: Иванов Иван Иванович, группа 18ПС114, 2011 г.

При выполнении всех вышеперечисленных требований и прохождении теста, вы получаете за экзамен 10 баллов.

Для получения оценки выше 10 баллов, на сайте должны использоваться веб-технологии, изученные вами самостоятельно, например:

  • скоростная оптимизация
  • поисковая оптимизация (SEO)
  • микроформаты
  • микроданные
  • условные комментарии для браузеров семейства Internet Explorer
  • CSS-стили для различных устройств: принтер, КПК, ТВ
  • JavaScript или другой клиентский язык программирования
  • PHP или другой серверный язык программирования
  • AJAX
  • Flash + ActionScript
  • Java
  • и/или другие (по согласованию с преподавателем)
